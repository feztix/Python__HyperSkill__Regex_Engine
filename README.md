## About The Project

<img src="https://i.ibb.co/164JDfH/image.png" alt="Logo">

This project is a part of the following track: <a href="https://hyperskill.org/tracks/2">Python Developer</a>

Regular expressions are a fundamental part of computer science and natural language processing. In this project, you will write an extendable regex engine that can handle basic regex syntax, including literals (a, b, c, etc.), wild-cards (.), and metacharacters (?, *, +, ^, $).

# Learnings Outcomes

1. Essential regex syntax
2. Practice working with parsing and slicing
3. Applcating boolean algebra and recursion

### Built With

* [Python 3.8](https://www.python.org/downloads/release/python-380/)
